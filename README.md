An AI-powered examination monitoring system that detects unethical student behaviour such as mobile phone usage, chit passing, peeking, copying from hand, and supplement passing using a custom-trained YOLO11L model.

---

## Overview

This project is a computer vision–based examination surveillance system developed to automate exam monitoring using Artificial Intelligence.

The system analyzes either:

- Live webcam/IP webcam feeds
- Pre-recorded examination videos

and detects suspicious behaviour in real time by drawing bounding boxes around detected objects while simultaneously generating timestamp logs for later review.

The project utilizes a custom-trained YOLO11L model trained on a self-created dataset consisting of multiple cheating scenarios captured from different viewing angles.

---

## Motivation

### The Problem

Traditional examination monitoring relies entirely on human invigilators.

This introduces several challenges:

- Difficult to monitor large examination halls
- Small cheating incidents often go unnoticed
- Human fatigue reduces monitoring accuracy
- Reviewing CCTV recordings manually is time-consuming
- Maintaining fairness during examinations becomes challenging

---

### The Solution

This project automates examination monitoring using Artificial Intelligence.

The system:

- Detects unethical behaviour in real time
- Highlights suspicious activities using bounding boxes
- Supports both webcam and recorded video analysis
- Generates timestamp logs for quick review
- Reduces dependence on continuous manual invigilation

## Model Weights

The trained YOLO11L model weights (`.pt`) are **not included** in this repository.

To run the project:

- Place your trained YOLO model inside the project directory.
- Update the model path in the source code if necessary.

Example:

```python
model = YOLO("your_model.pt")
```
